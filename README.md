Chinese
这是一个可以在浏览器中运行FFmpeg的库，它基于WebAssembly和Web Worker技术实现。
你可以部署在任何支持ES模块的HTTP服务器上，例如Nginx、Apache等。
也可以直接部署到任何支持ES模块的CDN，例如Cloudflare Workers、Vercel等。
不需要你上传任何媒体文件到服务器，即可实现使用网页对本地视频或音频文件进行处理。
比如对音频、视频进行切割、拼接、转码、压缩、调整音量等操作

英文
This is a library that can run FFmpeg in a browser, based on WebAssembly and Web Worker technologies.
You can deploy on any HTTP server that supports ES modules, such as Nginx, Apache, etc.
It can also be directly deployed to any CDN that supports ES modules, such as Cloudflare Workers, Vercel, etc.
You don't need to upload any media files to the server to process local video or audio files using web pages.
For example, performing operations such as cutting, splicing, transcoding, compressing, and adjusting volume on audio and video
